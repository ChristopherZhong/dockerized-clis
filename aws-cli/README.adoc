= AWS Command Line Interface (CLI)

https://aws.amazon.com/cli/

== Using the AWS CLI

Run the AWS CLI with the following command.

.Bash
[source,bash]
----
include::aws[lines=3..11]
	<command>
----

.CMD
[source,cmd]
----
include::aws.cmd[lines=3..11]
	<command>
----


Or use the link:aws[] or link:aws.cmd[] script.

[source,shell]
----
aws <command>
----
