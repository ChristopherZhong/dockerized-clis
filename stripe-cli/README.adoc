= Stripe Command Line Interface (CLI)

https://github.com/stripe/stripe-cli

== Using the Stripe CLI

Run the Stripe CLI with the following command.

.Bash
[source,bash]
----
include::stripe[lines=3..12]
	<command>
----

.CMD
[source,cmd]
----
include::stripe.cmd[lines=3..12]
	<command>
----

Or use the link:stripe[] or link:stripe.cmd[] script.

[source,shell]
----
stripe <command>
----
